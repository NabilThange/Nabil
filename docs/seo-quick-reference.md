## 📝 Blog Post Template

### Structure

```markdown
# [Title with Primary Keyword]

> Brief summary (40-60 words) - Answer-first format

## What is [Topic]?

Direct answer with keyword in first 100 words.

## Why [Topic] Matters

Benefits and use cases.

## How to [Action]

Step-by-step guide with examples.

## Best Practices

Expert tips and recommendations.

## Common Mistakes to Avoid

Troubleshooting and pitfalls.

## Conclusion

Summary and CTA.
```

### SEO Checklist for Blog Posts

- [ ] Title with primary keyword (60 chars max)
- [ ] Meta description with keyword + CTA (160 chars)
- [ ] H1 = Title
- [ ] H2s as questions or topics
- [ ] Keyword in first 100 words
- [ ] 1,500-2,500 words minimum
- [ ] 3-5 images with alt text
- [ ] 5-10 internal links
- [ ] 2-3 external authoritative links
- [ ] FAQ section at end
- [ ] Author bio
- [ ] Related posts
- [ ] Social sharing buttons

## 🤖 AI Optimization Quick Tips

### Princeton GEO Methods (Apply to All Content)

1. **Cite Sources** (+40%) - Link to official docs
2. **Add Statistics** (+37%) - Include specific numbers
3. **Add Quotations** (+30%) - Expert quotes
4. **Authoritative Tone** (+25%) - Confident language
5. **Improve Clarity** (+20%) - Simplify complex topics
6. **Technical Terms** (+18%) - Use domain vocabulary
7. **Fluency** (+15-30%) - Natural, readable content

### Answer-First Format

```markdown
## [Question]?

[Direct answer in 40-60 words with keyword]

[Detailed explanation follows...]
```

### Content Structure for AI

- Use bullet points and numbered lists
- Create comparison tables
- Add FAQ sections
- Include statistics with sources
- Use question-based H2 headings
- Keep paragraphs short (2-3 sentences)

## 🚨 Quick Fixes

### If Traffic Drops

1. Check Google Search Console for errors
2. Verify sitemap is accessible
3. Check for broken links
4. Review recent content changes
5. Check Core Web Vitals

### If Rankings Drop

1. Check competitor changes
2. Review content quality
3. Add more internal links
4. Update content with fresh info
5. Build more backlinks

### If AI Citations Drop

1. Update llms.txt
2. Refresh content (add dates)
3. Add more statistics
4. Improve answer-first format
5. Check AI crawler access

## 💡 Quick Tips

### Content Creation

- Write for humans first, search engines second
- Answer questions directly at the top
- Use examples and code snippets
- Include images and diagrams
- Add FAQ sections
- Update regularly (quarterly minimum)

### Technical SEO

- Keep site fast (< 3s load time)
- Ensure mobile-friendly
- Fix broken links immediately
- Update sitemap after new content
- Monitor Core Web Vitals weekly

### Link Building

- Focus on quality over quantity
- Engage authentically in communities
- Create valuable, shareable content
- Build relationships, not just links
- Track all backlinks

### AI Optimization

- Use answer-first format
- Add statistics and sources
- Structure content clearly
- Update llms.txt quarterly
- Monitor AI citations monthly
